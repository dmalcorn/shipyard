# Sandbox

Safe test files for demonstrating Shipyard's agent capabilities.

These files are **not** part of the production codebase. They exist so
you can point the agent at them for read, edit, and write demos without
risking real source files.

## Files

| File | Purpose |
|------|---------|
| `hello.py` | Python module with simple functions — good for surgical edit demos |
| `config_sample.yaml` | YAML config — good for demonstrating config edits |
